## CSS's Transform, Transition, and Animations

The CSS transform function is as follows:
```
div {
    transform: scale(1.5);
}

```

The transform feature follows this syntax:

```
.grow:hover
{
        -webkit-transform: scale(1.3);
        -ms-transform: scale(1.3);
        transform: scale(1.3);
}
```
